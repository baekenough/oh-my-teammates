# oh-my-teammates

Team collaboration addon for oh-my-customcode.

## Project Info

- **Language**: TypeScript (Bun runtime)
- **Version**: 0.2.0
- **License**: MIT
- **Peer Dependency**: oh-my-customcode >= 0.19.3

## Development

```bash
bun install
bun test
bun run build
```

## Git Workflow

| Branch | Purpose |
|--------|---------|
| `main` | Stable releases |
| `develop` | Development |
| `feature/*` | Feature branches from develop |

Conventional commits: `feat:`, `fix:`, `docs:`, `chore:`
